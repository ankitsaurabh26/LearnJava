## 🛠️ How to Run the Code
To run a Java program, you have to follow two main steps: Compiling & Running. 🏃💨
#### 1. Compile the Code 🧱
```
javac filename.java
```
- **What happens?** This creates a new file called `name.class` 📄
#### 2. Run the Program 🚀
```
java name
```
